# Sistema de Simulación

Aplicación académica desarrollada en Python con CustomTkinter para demostrar conceptos de simulación, probabilidades y modelos epidemiológicos.

## Características
- Login y registro de usuarios con SQLite.
- Dashboard moderno con tarjetas.
- Calculadora de simulación básica.
- Simulación COVID basada en estados.

## Requisitos
- Python 3.10+
- Instalar dependencias:

```bash
pip install -r requirements.txt
```

## Ejecución
```bash
python main.py
```

## Estructura del proyecto
- login.py: autenticación y pantalla de bienvenida.
- dashboard.py: panel principal de la aplicación.
- calculator_window.py: interfaz de la calculadora.
- simulation_calculator.py: lógica de simulación.
- database.py: acceso a SQLite.
- register.py: registro de usuarios.
